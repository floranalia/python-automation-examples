# Python Automation Examples

Small collection of Python automation scripts for API consumption, JSON processing and CSV validation.

## Contents
- `api_request.py` – Makes a request to a public API and logs the response.
- `json_processor.py` – Reads and transforms JSON data.
- `csv_cleaner.py` – Cleans and validates a CSV file.

## Technologies
- Python 3
- `requests` library
- JSON handling
- Basic data processing

These examples demonstrate foundational automation skills applied to common tasks.
